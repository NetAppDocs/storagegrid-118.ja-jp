= アラートの管理：概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アラートシステムでは、 StorageGRID の運用中に発生する問題を、使いやすいインターフェイスを通じて検出し、評価し、解決することができます。

カスタムアラートの作成、アラートの編集または無効化、アラート通知の管理を行うことができます。
